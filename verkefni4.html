Milling something huge!

The fourth project of the semester was part of a larger group project, involving milling an object of our liking. We wanted to create something functional and interesting and so the group decided on milling a 1:1 press fit bar stool.

We decided to divide the project among ourselves so each of us only oversaw a part of it. My contribution to the project involved creating the toolpaths for the stool parts. (The stool's design process can be found here.) During the course of this project I utilized only one program: VCarve.

VCarve is a software solution for cutting parts on a CNC Router. It allows you to produce 2D patterns with profile, pocket, drill and inlay toolpaths.

Below is the final layout of the design of the stool. The design was exported from Fusion 360 by a group member, Jóel Ingason, as a .DFX file for me to work with in VCarve.

When I had the .DXF file uploaded to VCarve, I was ready to go. The first thing I did was create the tool. When setting it up, I followed the corresponding video on the class website for instructions. As can be seen on the image below, the tool had the following settings: Tool Type set as End Mill; Spindle Speed set to 10500 rpm; Feed Rate set to 70 mm/sec; and Plunge Rate set to 50 mm/sec.

Next, I created the dog bone fillets. Because all individual parts are press fit together, the fit had to be tight for the stool to be functional and therefore the dog bone fillets are used to make sure the ends fit together properly and to make assembly possible.

When the dog bone fillets were complete I moved on to determining the best places to anchor the plywood board before milling. Below you can see the location of the screws relative to the layout of the model (left) as well the calculated CNC toolpath for the screws (right).

When the holes' location had been carefully determined I moved on to the tabs. The tabs are crucial to the milling process as they keep the parts in place and ensure an even cut along the defined toolpaths. The tabs can be seen in the below image, marked in green. The tabs are easily removed after milling.

Lastly, I created the actual toolpaths, both for the milling of inside cuts (left) and outside cuts (right).

Because I was limited to a trial license, as opposed to a pro licence, the finalization of the settings were set in the FabLab and the G codes exported.
